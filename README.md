## 📜 Description  

The $IOB token smart contracts are deployed across **five EVM-compatible blockchains**  
to ensure **scalability, accessibility, and interoperability** within the IO Beats ecosystem.  

Each contract follows the **ERC-20 / BEP-20 standard**, is **verified on-chain**,  
and has been **audited by [Cyberscope](https://www.cyberscope.io/audits/iob)** to ensure security and transparency.  

> 🛡️ All contracts are fully **open-source**, licensed under **MIT**,  
> and integrated with **Safe multisigs** for treasury and governance operations.
